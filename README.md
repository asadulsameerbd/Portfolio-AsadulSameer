## 📌 Features

* ✨ Modern and clean UI design
* 📱 Fully responsive (Mobile, Tablet, Desktop)
* ⚡ Fast performance
* 🎯 Projects showcase with details
* 🎥 Video preview support for projects
* 🏷️ Technology tags for each project
